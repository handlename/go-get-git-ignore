# go-get-git-ignore

Fetch and save .gitignore file from github/gitignore quickly.

## Usage

```sh
$ get-git-ignore --lang=Go
$ ls -a
.gitignore
```

## Installation

```sh
$ go get handlename/go-get-git-ignore
```
